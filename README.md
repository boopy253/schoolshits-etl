# School Shits ETL

```bash
shitsetl \
 --source docs/xxxx.xls \
 --template docs/template.xlsx \
 --output '2018 年春季年xx市中小学校教材、教辅采购情况统计表.xlsx' \
 --school "xxxx小学" \
 --year 2018 年春 \
 --start-row 4
```

shitsetl \
